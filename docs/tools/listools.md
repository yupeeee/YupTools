## argmax_list

Returns the index of the largest value in a given list.

```
max_idx: int = argmax_list(lst: List[Any],)
```

### Input

- **lst** (*list*):
List containing elements that can be compared by size, magnitude, etc.

### Output

- **max_idx** (*int*):
Index of the largest value in a given list.
